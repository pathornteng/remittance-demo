# Project summary

This demo can be used to demonstrate the money transfer using a token created by Stablecoin Studio.
Note: It supports ECDSA accounts only.

## Prerequisite

```bash
node
```

## Install

```bash
git clone git@github.com:pathornteng/remittance-demo.git
cd remittance-demo
npm install
```

## Run

```bash
npm start
```

## Build

```bash
npm run build
```
